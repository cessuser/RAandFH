Before the game starts in each period, the Firm will have the opportunity to roll a virtual die on the computer, which generates a random number from 1 to 6.
1)If the number on the die is 5 or 6, then in that period the Firm will receive a GOOD certificate only if s/he chooses High in that period, otherwise the Firm will receive a BAD certificate.
2)If the number on the die is 1, 2, 3 or 4, then the Firm will receive a GOOD certificate regardless of what s/he chooses in that period.
Once the Firm receives a BAD certificate, the BAD certificate will remain for the rest of the periods in the same block.

If you are the Firm, then …
Virtual Die Certificates
5,6 GOOD if you choose High BAD if you choose Low
1,2,3 GOOD no matter what


If you are the Investor, then …
Seeing a firm with certificate Meaning
GOOD The Firm has been complying with the above rule so far in the current block.
BAD The Firm violated the rule in some previous period in the current block.

Before the game starts in each period, starting in period 2, the Investor will receive a message telling them whether the Firm has a GOOD or BAD certificate so far in the current block.
